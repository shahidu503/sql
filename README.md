# SQL Portfolio: Data Cleaning & EDA

Welcome to my SQL repository! This project showcases my journey of learning SQL, practicing core database concepts, and applying them to clean and analyze data.

## 📚 What is in this Repository?

* **Foundational Queries:** Basics like SELECT, WHERE, GROUP BY, HAVING, and ORDER BY.
* **Advanced Concepts:** Practiced CTEs, Window Functions, Subqueries, Stored Procedures, and Triggers.
* **Data Cleaning:** Scripts showing how to handle missing data, fix formats, and remove duplicates.
* **Exploratory Data Analysis (EDA):** Scripts focused on extracting insights and analyzing data trends.

## 🛠️ Tools Used
* **Language:** SQL (Structured Query Language)
* **Focus:** Data Cleaning & Data Analysis
